# Japanese Style

## Tone

Default to polite written Japanese. Use です・ます for most ES unless the user asks for だ・である.

Good ES prose is:

- Clear
- Specific
- Calm
- Evidence-based
- Forward-looking

Avoid:

- Excessive humility: 「未熟ではありますが」repeated too often.
- Excessive confidence: 「必ず大きく貢献できます」without evidence.
- Translationese: 「私はこの経験から多くのことを学びました」without naming the learning.
- Generic enthusiasm: 「非常に魅力を感じました」「深く共感しました」without a reason.

## Natural Replacements

Use these replacements when revising:

```text
とても興味があります -> 強い関心を持っています
勉強しました -> 学びを深めました / 知識を身につけました
頑張りました -> 継続的に取り組みました / 改善を重ねました
いい経験になりました -> [specific learning]を学びました
コミュニケーション能力 -> 相手の状況を踏まえて調整する力
リーダーシップ -> 周囲を巻き込みながら前に進める力
```

## Sentence Control

Keep one sentence to one main point. If a sentence has more than two clauses, split it.

Prefer:

```text
当初は、メンバー間で作業の進め方に差があり、進捗が遅れていました。そこで私は、各自の担当範囲と期限を整理し、週に一度進捗を共有する場を設けました。
```

Avoid:

```text
メンバー間で作業の進め方に差があって進捗が遅れていたので、私は各自の担当範囲と期限を整理して週に一度進捗を共有する場を設けることで改善しました。
```

## Final Check

Check for:

- Same phrase repeated too often.
- First-person overuse: 「私は」at the beginning of every sentence.
- Unsupported claims.
- Company name and industry accuracy.
- Word or character limit.
